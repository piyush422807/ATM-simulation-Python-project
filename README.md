# ATM-simulation-Python-project
This is a simple ATM project made using Python that works like a basic banking system.

In this program, the user can check their balance, deposit money, withdraw money, and also see their transaction history. The account starts with a balance of ₹0, and every deposit or withdrawal is saved so the user can track all transactions.

The program also checks if the user has enough balance before allowing a withdrawal, which makes it work more like a real ATM.

This project is great for beginners because it helps in understanding Python concepts like functions, loops, lists, conditions, and user input in a practical and easy way.

It is a small console-based project, but it gives a good idea of how an actual ATM system works.
